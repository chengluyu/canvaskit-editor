Supported features:

  • Display the caret. Note: the caret does not blink for now.
  • Accept inputs from keys and input methods. Note: composition text does not show for now.
  • Select a range of text by mouse.
  • Delete characters with Delete and Backspace.
  • Delete selected text with Delete and Backspace.
  • Move the caret with arrow keys.
  • Double click to select words.
